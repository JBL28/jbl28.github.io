# How To
두 정수를 입력받아 합을 출력하는 문제입니다.<br>
변수 두개를 선언하고, scanf()함수를 이용해 입력받고, printf()함수를 이용해 출력합니다.<br><br>
# 소스코드
```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
	int A, B;
	scanf("%d%d", &A, &B);
	printf("%d", A + B);
	return 0;
}
```
<br>
# 입력
```cpp
4 5
```
<br>
# 출력결과
```cpp
9
```
