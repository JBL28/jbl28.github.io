# How To
[(문제 페이지 링크)](https://www.acmicpc.net/problem/1001)<br>
두 정수를 입력받아 두 수의 차를 출력하는 문제입니다.<br>
두 정수를 scanf() 함수를 이용해 입력받고 printf() 함수를 이용하여 출력합니다.<br>

# 소스코드
```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
	int A, B;
	scanf("%d%d", &A, &B);
	printf("%d", A - B);
	return 0;
}
```

# 입력
```cpp
3 2
```

# 출력
```cpp
1
```
